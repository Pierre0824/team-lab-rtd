iHealth Integration for Control-Plane Security
==============================================

F5 iHealth provides automated verification of control-plane security settings.

Capabilities
------------

* Security Best Practices Panel (QKView Security tab)
* Automated diagnostic heuristics
* Detection of misconfigurations and vulnerabilities
* Indicators of compromise

Specific Heuristics
-------------------

* D006068: Login banners
* D009908: Idle logout
* D012815: Appliance mode
* D015632: Admin/root account status
* H444724: Public access
* H494013: Password policy
* H726514: NTP servers

Automation Options
------------------

* Automate QKView uploads via BIG-IQ
* Schedule regular assessments
* Integrate with configuration snapshots
